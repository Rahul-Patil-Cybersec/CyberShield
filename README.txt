CyberShield – User Security Assessment & Awareness System
A Django-based Cyber Security Awareness and Assessment Platform. 

Final Year Academic Project | Developed by Rahul Patil

Requirements:
Python 3.10+
VS Code

Steps to run:

1. Download the project 

2. Open folder in VS Code

3. Open Terminal

4. Create virtual environment:
python -m venv venv

5. Activate venv:
venv\Scripts\activate

6. Install requirements:
pip install -r requirements.txt

7. Run migrations:
python manage.py migrate

8. Run server:
python manage.py runserver

9. Open browser:
http://127.0.0.1:8000/

Admin panel:
http://127.0.0.1:8000/admin

Project ready.
